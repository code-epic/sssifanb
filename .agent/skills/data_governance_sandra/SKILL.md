# Estrategia Sandra Server

- **PDF Inteligente**: El documento no es estático, es blindado. Cifrado constante `AES-256`.
- **Control Remoto Activo**:
  - Restricción estricta aplicada de Copiado/Pegado e Impresión.
  - Oscurecimiento contra capturas de pantalla integradas.
- **Trazabilidad Absoluta**: Auditoría real, caducidad temporal dinámica (autodestrucción virtual) y marcas de agua trazables.
- **Accesos**:
  - _Canal Privado_: App Sandra con biometría y autenticación sin contraseñas (`FIDO2`).
  - _Canal Público_: Validación "Zero-Trust" mediante QR para certificados inmediatos en la nube.
