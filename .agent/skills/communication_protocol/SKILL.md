# Protocolo Estricto de Interacción IA

1. **Idioma**: Todas las respuestas lógicas, comentarios de código y commits deben ser enteramente en ESPAÑOL.
2. **Respuesta Expresa**: Ejecuta directamente el código requerido. Elimina saludos, disculpas redundantes, e informaciones web agregadas no solicitadas que desvíen de la tarea directa.
3. **Peligros Estructurales**: Detente inmediatamente ante parámetros ambiguos que alteren infraestructura general o bases de datos sin aprobación directa, y pide a un desarrollador humano asistencia explícita en ese paso.
4. **Rescate de Memoria Pura**: En caso de un crash contextural o incongruencias en el flujo que desbincule pasadas operaciones, responde con la alerta de error exacta a inyectar al usuario: "He perdido el contexto de la tarea actual, por favor aclara el objetivo".
